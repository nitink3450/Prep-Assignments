Q.3.What is Hoisting?

Hoisting :Hoisting is a Javascript Mechanism where variables and function declarations are moved
to top of there scope before the code execution.
It is a default behaviour of js and it allows us to access a variable before initiallization.
e.g.: var
let and const aslo goes under hoisting but due to TDZ they behave differrently.
Hoisting supports only  Declarations not the initilizations
